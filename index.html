SKILLS

HTML

レスポンシブを含め、命名規則を意識したセマンティックなコーディングを心がけています。Sassも少しですが学びました。

CSS

レスポンシブを意識したスタイルを記述することができます。更新のしやすさを意識して、一定の規則性を持たせたコーディングを心がけています。Sassも少しですが学びました。

JavaScript

jQuery、およびjQueryプラグインを設置することができます（slick,bxslider,Lightbox2　など）

Photoshop

主に画像のレタッチ、バナーの作成に使用しました。基本操作は問題なく行うことができます。なるべくすべての動作をショートカットキーで行えるよう意識して使っています。

Illustrator

主にwebサイトに使用する素材やロゴの作成に使用しています。基本操作は問題なく行えます。ペンツールをもっと使いこなせるよう練習中です！

After Effect

簡単なアニメーションやプリセットの実装等の簡単な基本操作が可能です。

XD

ワイヤーフレームやデザインカンプの作成に使用しています。

Figma

XDより使用頻度が少ないですが、基本操作は問題ありません。

WordPress

サーバー環境、ローカル環境(XAMPP)ともにWordPressのインストール・設定ができます。テーマのカスタマイズや、CSSによるデザインの変更もできます。

Git / Github

コミットなど、基本の操作は行うことができます。

Excel

表の作成や簡単な関数は使いこなせます。普段は家計簿や履歴書を作成する際に使用しております。

Word

職務経歴書などの書類を作成する際に使用しております。

VisualSutudioCode

拡張機能の導入やwebサイトの実装で利用しています。

ChatGPT

コードの修正やアイデア出しなどに使用しています。
